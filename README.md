# ご覧いただきありがとうございます。
pytweetはSNSアプリケーションです。

## 環境
HTML/ CSS/ Python/ Flask/ MySQL(phpMyAdmin) を使用しています。

# 機能
 登録/ログイン、
 文章と写真のアップロード、
 ツイートに対しての返信、
 フォロー/フォロー解除、
 ツイート、返信の編集・削除、
 いいねボタン、
 ユーザー検索。
